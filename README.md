# Mandelbrot Explorer
 * A Unity Editor to explore the Mandelbrot fractal.
 * User can change the color gradient, zoom in, zoom out and rotate the fractal.
 ## Controls
 * W - UP
 * A - LEFT
 * S - DOWN
 * D - RIGHT
 * Q - ROTATE LEFT
 * E - ROTATE RIGHT
 * MIDDLE MOUSE BUTTON SCROLL - ZOOM IN AND OUT
